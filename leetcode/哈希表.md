## 有效的字母异位词
https://leetcode.cn/problems/valid-anagram/description/

```java
public boolean isAnagram(String s, String t) {
    if(s==null || t==null) return false;
    if(s.length()!=t.length()) return false;

    Map<Character, Integer> map = new HashMap();
    char[] cs = s.toCharArray();
    for(int i = 0;i < s.length(); i++){
        char c = cs[i];
        Integer v = map.get(c);
        if(v== null) map.put(c,1);
        else map.put(c,v+1);
    }

    cs = t.toCharArray();
    for(int i = 0;i < s.length(); i++){
        char c = cs[i];
        Integer v = map.get(c);
        if(v==null) return false;
        v = v-1;
        if(v<0) return false;
        map.put(c,v);
    }

    return true;

}

// 优化点：
// 因为key的范围只有 字母 ，即有限对象池，使用数组来进行对比;直接使用 charAt 来获取对应的字符
public boolean isAnagram(String s, String t) {
    if(s==null || t==null) return false;
    if(s.length()!=t.length()) return false;

    int[] record = new int[26];
    for(int i = 0;i< s.length();i++){
        record[s.charAt(i)-'a']++;
    }
    for(int i = 0;i< s.length();i++){
        record[t.charAt(i)-'a']--;
        if(record[t.charAt(i)-'a']<0) return false;
    }

    return true;
}
```

## 两个数组的交集
https://leetcode.cn/problems/intersection-of-two-arrays/description/

```java

public int[] intersection(int[] nums1, int[] nums2) {
    if(nums1 == null || nums2 == null) return null;
    Set<Integer> record = new HashSet();
    Set<Integer> result = new HashSet();

    for(int i=0;i<nums1.length;i++){
        record.add(nums1[i]);
    }

    for(int i=0;i<nums2.length;i++){
        if(record.contains(nums2[i])){
            result.add(nums2[i]);
        }
    }

    int[] array = new int[result.size()];
    int index = 0;
    for (int num : result) {
        array[index++] = num;
    }
    return array;
}
```

## 四数相加
https://leetcode.cn/problems/4sum-ii/
```java
public int fourSumCount(int[] nums1, int[] nums2, int[] nums3, int[] nums4) {
    Map<Integer,Integer> record = new HashMap();

    for(int num1 : nums1){
        for(int num2 : nums2){
            int sum = num1+num2;
            if(record.containsKey(sum))
            record.put(sum,record.get(sum)+1);
            else
            record.put(sum,1);
        }    
    }

    int count = 0;

    for(int num3 : nums3){
        for(int num4 : nums4){
            int sum = num3+num4;
            if(record.containsKey(0-sum))
                count+=record.get(0-sum);
        }    
    }

    return count;
}
俩俩一组 O(n^2) O(n)
```
